[https://github.com/14-Prasanna/Prasanna_Portfolio](https://github.com/14-Prasanna/Prasanna_Portfolio)

## 🚀 About the Project

This portfolio is built with a modern web stack to ensure performance, responsiveness, and clean code architecture.

### Built With:
- **React 19**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion & GSAP** (for smooth scrolling and micro-animations)
- **TanStack Router**

## ✨ Features
- **Responsive Design:** Optimized for seamless viewing on all devices.
- **Dynamic Animations:** Engaging hero section and journey timeline powered by GSAP and Framer Motion.
- **Project Showcase:** Detailed cards for my automation projects involving Selenium, Pytest, Rest Assured, BDD, and CI/CD pipelines.

## 💻 Getting Started Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/14-Prasanna/Prasanna_Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Prasanna_Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
